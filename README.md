# üá∫üá∏ Project description
The Commerce project is a web application developed with React.js and Next.js. It allows users to browse a list of products, add and remove items from the shopping cart, and view the added items on a dedicated cart page.

# Features

## Product list
- Displays a list of products fetched from an external API.
- Each product shows an image, title, and price.
- Button to add or remove the product from the cart.

## Shopping cart
- View the products added to the cart.
- Option to remove a product one at a time.
- Option to remove all products at once.
- Cart icon in the header with the count of added items.

# Project setup and execution

- npm

## Install dependencies:

## Run the application:
